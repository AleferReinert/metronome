# Metronome

## Features
- Dark and light theme
- Time signatures 2 ~ 12 / 2, 4, 8, 16
- 30 to 280 bpm
- Beat counter
- Lights
- Separate volume control for quarter, eighth and triplet notes
- 4 options for accent and click sounds
- Option to play only odd beats
